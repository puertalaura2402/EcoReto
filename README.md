# EcoReto 
Aplicación web gamificada desarrollada en ASP.NET que promueve hábitos sostenibles mediante retos ecológicos.

## Tecnologías
- ASP.NET MVC
- C#
- SQL Server
- HTML, CSS, Bootstrap

## Funcionalidades
- Sistema de usuarios
- Retos ambientales
- Seguimiento de progreso

## Autoras del proyecto
- Laura Puerta Arboleda
- Mariana Muñoz Chaparro

Proyecto desarrollado como parte de la tecnología en desarrollo de software.

## Cómo ejecutar el proyecto
1. Clonar el repositorio:
   git clone https://github.com/puertalaura2402/EcoReto.git

2. Abrir la solución en Visual Studio:
   EcoReto.sln

3. Configurar la base de datos en SQL Server

4. Ejecutar la aplicación desde Visual Studio
